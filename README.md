# Memrynote Tap

## Install Memry

```sh
brew install --cask memrynote/tap/memry
```

Or:

```sh
brew tap memrynote/tap
brew install --cask memry
```

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "memrynote/tap"
cask "memry"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
